## ✨ Tentang Proyek

Proyek ini adalah halaman depan (landing page) resmi untuk **Dthree Studio**, sebuah studio foto profesional yang berlokasi di Palangka Raya. Website ini dirancang dengan antarmuka yang modern, bersih, dan responsif, menonjolkan estetika hangat dan elegan dari studio.

## 🚀 Fitur Utama

- **Desain Modern & Estetik**: Tema visual *glassmorphism* dan warna-warna hangat (`#FDFBF7`, `#4A3728`, `#634832`).
- **Antarmuka Responsif (Mobile-Friendly)**: Tampilan sempurna di semua ukuran layar (Desktop, Tablet, Mobile).
- **Navigasi Mulus**: Efek navigasi *sticky* dan animasi yang interaktif.
- **Galeri & Pricelist**: Pengaturan rapi untuk menampilkan portofolio dan paket harga yang ditawarkan.
- **Integrasi Peta & Kontak**: Akses langsung ke WhatsApp CS, Instagram, dan Google Maps lokasi studio.

## 💻 Tech Stack

- **HTML5**: Struktur halaman website.
- **CSS3**: Kustomisasi gaya untuk antarmuka.
- **JavaScript**: Interaktivitas untuk menu mobile, slider, pop-up galeri, dan *smooth scrolling*.
- **FontAwesome**: Ikon-ikon minimalis dan modern.

## 💻 Untuk Developer...

Project ini sangat mudah untuk dikustomisasi, jadi orang awam dapat mengedit, menambah, dan menghapus beberapa isi fitur didalamnya secara mudah dan bebas.

## 🛠️ Cara Menjalankan

Karena proyek ini menggunakan HTML, CSS, dan JavaScript murni, Anda tidak memerlukan instalasi server atau dependensi khusus.

1. **Clone** repository ini atau unduh *source code*.
   ```bash
   git clone https://github.com/epennotgg/dthree-studio-landing-page
   ```
2. Buka folder proyek.
3. Klik ganda pada file `index.html` untuk menjalankannya di browser web pilihan Anda (Chrome, Firefox, Safari, Edge, dll.).

*(Opsional)* Anda dapat menggunakan ekstensi **Live Server** di VS Code untuk pengalaman *development* yang lebih baik:
- Buka folder proyek di VS Code.
- Klik kanan file `index.html` -> Pilih **"Open with Live Server"**.
